# 💘 Dating Bot — Telegram Bot для Закарпаття

> Анонімний бот знайомств з веб-адмін-панеллю та системою модерації

---

## ✨ Можливості

| Модуль | Опис |
|--------|------|
| 👤 **Реєстрація** | Вибір статі та району Закарпаття |
| 🔍 **Пошук** | Анонімний пошук співрозмовника за параметрами |
| 💬 **Чат** | Анонімне спілкування 1‑на‑1 |
| ⚙️ **Анкета** | Зміна параметрів у будь-який момент |
| 📢 **Підписка** | Перевірка підписки на канали перед використанням |
| ⚠️ **Скарги** | Скарги на порушників із системою модерації |

---

## 🛡️ Адмін-панель

### 🤖 Через Telegram-бота

| Функція | Опис |
|---------|------|
| 📊 Статистика | Кількість користувачів, повідомлень, скарг |
| 👥 Блокування | Блокування / розблокування користувачів |
| ⚠️ Скарги | Прийняття або відхилення скарг |
| 📢 Розсилка | Масова розсилка повідомлень |
| 📺 Канали | Додавання / видалення каналів для підписки |
| 👑 Адміни | Управління адміністраторами *(тільки супер-адмін)* |

### 🌐 Веб-адмінка — `http://localhost:3000/admin`

- 📊 Дашборд зі статистикою в реальному часі
- 💬 Перегляд чатів між користувачами
- 👥 Список усіх користувачів
- ⚠️ Модерація скарг
- 👑 Управління адміністраторами

---

## 👑 Ролі адміністраторів

| Роль | Можливості |
|------|------------|
| 🔱 **Супер-адмін** | Повний доступ + управління адмінами |
| 🛠️ **Адмін** | Усі функції, крім управління адмінами |

## 🗄️ Структура бази даних (MongoDB)

```
📦 bot_db
 ┣ 👤 users      — анкети, статус блокування
 ┣ 💬 messages   — усі повідомлення в чатах
 ┣ ⚠️  reports   — скарги (статус, причина)
 ┣ 👑 admins     — адміни (роль: admin / super_admin)
 ┗ 📺 channels   — канали для обов'язкової підписки
```

## 🚀 Швидкий старт

### 1. Клонування репозиторію

```bash
git clone https://github.com/your-username/dating-bot.git
cd dating-bot
```

### 2. Встановлення залежностей

```bash
npm install
```

### 3. Налаштування `.env`

Створіть файл `.env` у корені проекту:

```env
# ── Telegram ──────────────────────────────────
BOT_TOKEN=ваш_токен_від_BotFather

# ── MongoDB Atlas ─────────────────────────────
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/bot_db

# ── Адмін-панель ──────────────────────────────
ADMIN_PASSWORD=ваш_надійний_пароль
FIRST_ADMIN_ID=ваш_telegram_id

# ── Сервер ────────────────────────────────────
PORT=3000
WEBAPP_URL=http://localhost:3000/admin
```

### 4. Створення супер-адміна

```bash
node add-superadmin.js
```

### 5. Запуск

```bash
# Термінал 1 — Telegram-бот
node index.js

# Термінал 2 — Веб-адмінка
node server.js
```

> Після запуску веб-панель доступна за адресою: **http://localhost:3000/admin**
