# 🤖 Dating Bot - Бот знайомств для Telegram

Бот знайомств для жителів Закарпаття з анонімним чатом, адмін-панеллю та системою модерації.

## ✨ Функції

- 👤 Реєстрація з вибором статі та району
- 🔍 Пошук співрозмовника за параметрами
- 💬 Анонімний чат між користувачами
- ⚙️ Налаштування анкети
- ⚠️ Система скарг на користувачів
- 🛡️ Адмін-панель з модерацією
- 📊 Статистика та моніторинг
